= Velero Workload

== Overview

This is a workload for Velero which is a backup / restore tool for Kubernetes. Following section explains how to install this workload on your OCP cluster.

=== Purpose

Velero is used as a backup / restore tool by RedHat OCP Migration Controller. Migration Controller migrates applications from one OCP cluster to the another.

This workload is used for a demo migration.
